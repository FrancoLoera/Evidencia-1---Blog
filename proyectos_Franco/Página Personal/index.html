Perspectiva del desarrollo web

Como lo mencioné anteriormente, actualmente trabajo de desarrollador web, por lo que he visto cosas muy interesantes. Antes para mí el desarrollo se limitaba a HTML, CSS, JS y PHP, pero conforme te vas adentrando a las diversas necesidades del mercado, puedes ver que las tecnologías implicadas en el desarrollo de un sitio web son muy diversas, esto debido a que no todas las páginas ni necesidades son las mismas, algunos clientes ocupan un e-commerce, otros una página meramente informativa, y otros pocos ocupan un airbnb. También, ver que la mayoría de las veces los proyectos no necesitan de un desarrollo enfocado en código es interesante, cambia mucho tu perspectiva ver las herramientas que realmente se utiliza y ver qué es lo que realmente genera valor en una página web (como el SEO, SEM y estrategias de marketing). El desarrollo web es muy interesante porque es demasiado amplio, me gusta descubrir poco a poco sobre él
